Capstone Project

This project is meant to showcase the use of natural language processing (NLP) with song lyrics. Language alone, can not capture the feel of any individual song but here I group songs based on the similarity in the lyrics to form the different “vibes”.

How it works:

  1. BERT - A pre trained BERT model encodes the input corpus, which here is a list of song lyrics
  2. KMeans - The output vectors from BERT are then grouped into clusters via KMeans
  3. Vibes - A WordCloud from the each cluster is created based on the cluster's song lyrics
  4. Spotify - The Spotify API is called on a random song when user clicks a vibe
